# 日志格式优化

## 概述

优化日志列表的显示格式，使用方框样式包裹时间和级别。

## 显示格式

每行日志包含三个部分：
- **时间框**：灰色背景，`MM-DD HH:mm:ss` 格式
- **级别框**：根据级别显示不同颜色
- **日志内容**：等宽字体，自动换行

## 级别颜色

| 级别 | 背景色 | 文字色 |
|-----|-------|-------|
| ERROR | 红色 | #ef4444 |
| WARNING | 橙色 | #f59e0b |
| INFO | 蓝色 | #3b82f6 |
| DEBUG | 紫色 | #8b5cf6 |
